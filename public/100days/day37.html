Hardware hacking (continued)

I connected the UART pins to a SALAE logic analyser but didn’t find any data as I had hoped for, only a tri state clock (later read something about connecting the Tx pin to the VCC but I haven’t tested that) but I did get to know the baud rate was 57600

So I decided to connect the UART pins to the bus pirate FTDI UART to Serial convertor.

MISO => Tx

MOSI => Rx

Gnd => Gnd (I just hooked it up to the RF shielding)

I used a CLI program called "screen" to interface with the hardware

sudo screen /dev/ttyUSB0
