NCL — Steganography

Decided to do some forensic work today. I had an old challenge lying from NCL’s fall game that I never ended up trying.

Steganography is the technique of hiding secret data within an ordinary, non-secret, file or message in order to avoid detection; the secret data is then extracted at its destination.

Example, these NCL challenges have .bmp files that contain secret messages that we need to recover to complete the challenge

STEG1.bmp

I installed the Digital Invisible Ink Toolkit (DIIT) and ran it using the following command:

java -jar diit-1.5.jar

Then I select the image to decode (STEG1.bmp) and an algorithm to use. There’s no way to know which algorithm was used to encode the information so I just brute all the algorithms till I get the answer. Battlesteg worked.

Flag: SKY-STEG-9418

STEG2.bmp

Same process, more number of tries. Blind-Hide worked.

Flag: NCL-UZFG-4614
