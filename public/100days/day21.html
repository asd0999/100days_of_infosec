Wargames —  leakme shorter

This program runs the same way as the previous leakme program, except that it only accepts 9 characters in the fgets ( ) function which means only 8 characters because the 9th character is a null byte

So that essentially means I cannot give the program the following input :  

%p%p%p%p%p%p%p%p%p%p%p%p%p%p%p%p%p%p



A way to get around this is to leak the exact value by using the following format :

%n$p //where n is the number of address I want to read from

So if I want to read from the 4th address, I would write :

%4$p



I found the solution of leakmeshorter by counting the number of leaked values from the previous challenge and a few trial and error.

The following input reveals the magic number!

%15$p
