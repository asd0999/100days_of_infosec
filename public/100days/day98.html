Bash scripting (part 5)

Filename substitution || Globbing

$ ls -1

$ ls *.mp4

$ ls (*.mp4|*.pdf)

$ ls [0-9](*.mp4|*.pdf)

$ ls [!(0-9)](*.mp4|*.pdf)

$ ls [!(0-9)]*765*pdf

$ ls (*765*pdf|*765*mp4)

$ ls [!(0-9)](*765*pdf|*765*mp4)

$ ls [0-9]?10*.(gif|mp4)

Target file: i-765.pdf

Also: [!(a-z)] is same as [^a-z]

If using regular expressions in code, you will need to escape special characters by using the back slash ‘\’

So . will become \. or { will become \{

This is a comprehensive set of all valid characters for filenames, i think

b-zA-Z0-9\-_\+\=\\\'\"\{\[\}\]

Some interesting examples:

M(r|s|rs)\.?\s[A-Z]\w*
