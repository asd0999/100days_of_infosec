Violent Python —  vulnScanner with arguments

I worked on the vulnScanner program to accept more arguments

1) Mode >>> 1 for scanning a single IP or 2 for a range

2) IP >>> for mode 1, mention an IP address; for mode 2 mention the first 24 bits/3 groups of an IP address"

3) Filename >>> a vuln banner file name to compare the scanned the IP/port banners against

Arguments are helpful so I don’t need to go into the program and write the IP addresses one wants to scan, I can just pass it as an argument

But what about if I want to scan just a single IP vs all 255 IPs in a a certain IP address range. So for that a designated the first argument (as mode). Depending on the mode the program will behave differently.

Mode 1 will scan only the mentioned IP address while mode 2 will scan all IPs in the 24bit IP address range. Also added a message to show when not enough arguments are provided to tell the user how to use the program effectively
