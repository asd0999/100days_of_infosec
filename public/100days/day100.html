System enumeration using Metasploit

First of all some good news, I passed my CompTIA Security+ certification!!!!

I’ve been looking into the Metasploit framework and wanted to make a post on some things I learnt about using a database with Metasploit. So here you go:

Kali > terminal

$ msfdb init

$ service postgresql start

$ msfconsole

msf> db_status (you will see [*]postgresql connected to msf)

msf> nmap -Pn -sS -A -oX Test 10.10.10.0/24

After the scan is complete...

msf> db_import Test  (if you run db_nmap, don’t need to import)

msf> hosts (lists hosts in the db)

msf> services (to list all services running on all machines in the db)

msf> use scanner/smb/sms_version

msf auxiliary(scanner/smb/smb_version)> show options

set RHOSTS 10.10.10.8-16, set THREADS 100, run

msf auxiliary(scanner/smb/smb_version)> hosts 

→ to see compete host name, OS name, OS flavour info which previously wasn’t known

To export database, db_export -f xml -a ~/Desktop/mod2_lab5.xm

Note: the output file of the nmap scan “Test” will already be in your home folder (remember we had to db_import it into msf)
