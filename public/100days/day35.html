ccrypt - encryption is a two way function

So I was looking for an encryption tool for myself, preferably one that works off of the command line,  and I found ccrypt to work great!

ccrypt is a utility for encrypting and decrypting files and streams. It was designed to replace the standard unix crypt utility, which is notorious for using a very weak encryption algorithm. ccrypt is based on the Rijndael block cipher and is believed to provide very strong cryptographic security

Install ccrypt:

sudo apt install -y ccrypt (Linux)

sudo brew install ccrypt (for MAC)

Create file:

nano test.txt

echo “hey” > test.txt

To encrypt:

ccrypt test.txt

Enter key: ********

ls will show you a test.txt.cpt file

cat test.txt.cpt gives back gibberish

To decrypt:

ccrypt -d test.txt.cpt

ls will show you  the original test.txt file
