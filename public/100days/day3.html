vault-door-6

I had to google what ‘^’ operator means in java. It means XOR.

 

Important XOR property :

If A ^ B = C, then A = B ^ C

Applying this to the first myBytes[0] 

I wrote a quick solver in java to XOR the myBytes[values] with 0x55 to retrieve the flag.

Flag: picoCTF{n0t_mUcH_h4rD3r_tH4n_x0r_0c3a2de}

vault-door-7

In this chal, the java program converts the 32 character long password into an array of 8 integers (each integer is 4 bytes, 4*8=32) and then checks for a true return condition against those integers.

I could have used an online integer to binary converter and then a binary to ASCII text converter to retrieve the flag but a) that would be slow b) I wanted to use this as an opportunity to explore pwntools. So I wrote a short python script to do the work for me
