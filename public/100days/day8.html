Setting up Ubuntu 18.04 on a Windows machine

I recently got a Windows machine so I could dual boot it with Linux. Here is the process I followed to set up the dual boot as well as installing essential tools using the apt packet manager

Part I - Dual boot

Partition the hard drive/SSD

  • Disk management > Shrink OS  

Get ubuntu

  • plug in thumbdrive with ubuntu 18.04

BIOS setup

  • Restart machine and keep hitting hit F2 till the BIOS shows up
  • In some machines, you gotta press ESC
  • BIOS stands for Basic Input output system which is basically the system config

Disable Secure boot

  • Advanced mode > Security > Secure boot  > Disable

Dual Boot

  • Boot options (F8) > Select ubuntu
  • Configure the installation setup, click next, next, next
  • Check all the auto update stuff off

When I restarted the machine, I was prompted to select which OS I want to boot. Boom!
