CTFd —  boffin

Simple buffer overflow challenge, program contains a give_shell function. Need to write the address of the first instruction of the giver_shell function in the return address space on main’s stack

So I tried to send the following payload :

python -c "print'A'*40 + '\x9d\x06@\x00\x00\x00\x00\x00'" | ./boffin

But it did not work

Well, it did kinda work cause in gdb we can see that the rip reaches the correct address in the give_shell function

but the program still crashes, probably something to do with the function prologue and the fact that it is being called from main( )

So instead of overflowing the ret addr to the first instruction in the give_shell function, lets try

overflowing the ret addr to the first instruction after the function prologue

ie the instruction that starts setting up the parameters for the system call to run execve

python -c "print'A'*40 + '\xa1\x06@\x00\x00\x00\x00\x00'” | ./boffin

yes that worked!!! ok cool!!! :)
