Docker - II

So! Docker is a whole world of its own. I tried some stuff by following this tutorial :

https://rominirani.com/docker-tutorial-series-a7e6ff90a023

Then made this dockerfile for launching a containerized wireshark

FROM alpine:3.9

RUN apk add --update wireshark dbus font-noto

CMD wireshark

 

dockercompose.yml for setting the right permissions

version: '3'

services:

 wireshark:

   build: .

   image: d.bigj.dev/wireshark:latest

   container_name: wireshark

   privileged: true

   volumes:

     - "./data:/root:rw"

     - "/tmp/.X11-unix:/tmp/.X11-unix"

   environment:

     - DISPLAY=$DISPLAY

   network_mode: "host"

 

Thanks to big_j for the docker image!
