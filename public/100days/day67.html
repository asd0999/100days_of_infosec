CTFd —  go here 4 shell (part 1)

Multi stage web challenge which took me a couple days to do. First, I saw a login page so I tried SQL injection which didn’t work

So then I tried to leak the login source using php filters

I used the following parameter to successfully leak the base64 encoded source

?page=php://filter/convert.base64-encode/resource=login

I also tried to see if there’s an admin page and bingo, there was one. I leaked the source for that too using the same technique

From the source, I understood that the admin page redirects me to the welcome page unless the $_SESSION[‘id’] == 1

For the $_SESSION[‘id’] to be set to 1, I need to login as the admin (Im guessing thats the first entry in the db, hence id=1)

But SQL didn’t work and I was running out of ideas, so I thought I’d try to brute force it using a simple wordlist. Enter Burp.

But that didn’t work
