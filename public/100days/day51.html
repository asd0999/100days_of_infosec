Let’s encrypt

Let's Encrypt is a free, automated, and open certificate authority that one can use to register a certificate on their website/server

The I was using on my avakya web app expired and so I thought I’d try lets encrypt

It is super easy to use and the instructions provided on their website are super helpful.

https://letsencrypt.org/getting-started/

Since I have shell access I clicked on the certbot link and clicked on get instructions

https://certbot.eff.org/

https://certbot.eff.org/instructions

The instructions:

sudo apt update

sudo apt install software-properties-common

sudo add-apt-repository universe

sudo add-apt-repository ppa:certbot/certbot

sudo apt update

I used the standalone version to verify my server (you need to stop the server from running to do this)

sudo apt install -y certbot

sudo certbot certonly --standalone

And that’s it! Pretty simple!
